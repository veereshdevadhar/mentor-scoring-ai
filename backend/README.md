FastAPI backend for AI-powered mentor evaluation system.

## Tech Stack

- **Framework**: FastAPI
- **Server**: Uvicorn
- **Database**: MongoDB Atlas
- **AI Pipeline**:
  - Whisper (Transcription)
  - LLaMA 3.1 8B via Ollama (NLP Analysis)
  - MediaPipe (Visual Analysis)
  - Librosa + PyDub (Audio Analysis)
